‚ö° GC Algorithms & Stop-The-World Events

üîç Mark & Sweep

Marks live objects, sweeps dead ones. Used in CMS, Serial GC.

üì¶ Copying (Young Gen)

Copies live objects to another space. Used in Young Generation GC.

üóúÔ∏è Mark-Compact

Marks, then compacts live objects. Used in Old Generation GC.

‚è∏Ô∏è Stop-The-World (STW) Events:

STW pauses occur when GC stops all application threads. Minor GC (Young Gen) has short STW pauses (milliseconds). Major/Full GC (Old Gen) has longer STW pauses (can be seconds). Modern GCs (G1, ZGC, Shenandoah) minimize STW duration.

üî§ String Constant Pool Details
  • Location: Heap Memory (Java 7+), PermGen (Java 6)
  • Stores unique string literals
  • String.intern() adds to pool
  • String s = "Hello" ‚Üí pool
  • String s = new String("Hello") ‚Üí heap
  • Garbage collected like other objects
üéØ Memory Allocation Flow
  • Objects created in Eden Space
  • Minor GC moves survivors to S0/S1
  • Objects age with each survival
  • Promoted to Old Gen after threshold
  • Major GC cleans Old Generation
  • Full GC cleans entire heap

üí° String Pool vs Heap

üöÄ TLAB (Thread Local Allocation Buffer)
  • Small region in Eden per thread
  • Fast allocation without locks
  • Reduces thread contention
  • Enabled by default in JVM
  • -XX:+UseTLAB flag
  • Objects first try TLAB allocation

üí° Key Concepts

Heap: Shared among all threads, stores objects and arrays
Stack: Per-thread memory, stores method frames and local variables
Metaspace: Stores class metadata, grows automatically (Native memory)
String Pool: Heap area storing unique string literals (Java 7+)
PC Register: Program counter holding current instruction address
Native Method Stack: Supports JNI and native C/C++ methods
Minor GC: Cleans Young Generation (fast, frequent)
Major GC: Cleans Old Generation (slower, less frequent)
Full GC: Cleans entire heap (slowest, stop-the-world)
Runtime Constant Pool: Per-class constants in Metaspace
Direct Memory: Off-heap memory for NIO operations
TLAB: Thread-local allocation buffer for fast object creation
